# 📊 Excel Sales Analytics Project

A Python-based data processing and reporting system using **openpyxl**.

This project automates the analysis of sales data and generates a structured Excel report with multiple insights.

---

## 🚀 Features

- Load and process Excel dataset
- Compute key metrics:
  - Total Sales
  - Sales by Region
  - Product Performance
  - Customer Insights
  - Average Delivery Time
- Identify:
  - Top Product (overall)
  - Top Region
  - Top Products per Region
- Generate automated Excel report with multiple sheets

---

## 🧠 Tech Stack

- Python
- openpyxl
